## Описание игры

Это минималистичный 3D-раннер, в котором игрок (куб) автоматически движется вперёд по дороге. Задача — уклоняться от препятствий, собирать бонусы и выживать как можно дольше. При столкновении с препятствием игрок теряет здоровье. Когда здоровье заканчивается — игра автоматически перезапускается.

---

## Управление

| Клавиша | Действие |
|---------|----------|
| A / ← | Движение влево |
| D / → | Движение вправо |
| W | Прыжок |

---

## Реализованные механики

### Игрок
- Автоматическое движение вперёд
- Перемещение влево/вправо по трём полосам
- Прыжок (одинарный, с проверкой на землю)
- Здоровье (максимальное и текущее)
- Получение урона от препятствий
- Смерть при достижении 0 здоровья и перезапуск сцены

### Препятствия
- Два типа препятствий (красный куб — урон 1, синий составной — урон 2)
- Случайная генерация перед игроком
- Случайный выбор полосы (лево, центр, право)
- Препятствия удаляются, когда остаются позади

### Интерфейс
- Отображение текущей скорости на экране
- Отображение текущего здоровья
